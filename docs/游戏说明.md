# Rhythm Guard（节奏守卫）

> 版本：1.0.0  
> 类型：音乐节奏 × 合成塔防  
> 平台：WebGL HTML5（浏览器 / 单文件 HTML）  
> 线上试玩：https://naanjing0625-ship-it.github.io/rhythm-guard/

---

## 一、核心玩法介绍（200 字内）

音乐节奏 × 合成塔防。先在节奏关按圈色操作：黄圈单击、蓝圈长按、红圈连击，凭 Perfect/Good 评级获得道具；再在 6×6 棋盘部署并合成升级；最后自动塔防，守卫核心。善用对空与减速道具，用音律币修复核心，清空波次即可胜利。

---

## 二、游戏概述

《Rhythm Guard》将 **节奏判定**、**棋盘合成** 与 **塔防守卫** 串联为一局完整流程。节奏打得越好，获得的道具越多、品质越高；部署阶段决定阵容与合成路线；守卫阶段则考验对空、破甲与核心修复的综合策略。

**一局标准流程：**

```
主菜单 → 选择关卡 → 节奏阶段 → 部署阶段 → 守卫阶段 → 结算
```

**章节与关卡：**

| 章节 | 名称 | 关卡 |
|------|------|------|
| 第 1 章 | 王国乐章 | L01～L05 |
| 第 2 章 | 暗影森林 | L06～L07（需累计星级解锁） |
| 第 3 章 | 雷霆高地 | L08 终极守护（需累计 18 星解锁） |

首次进入建议先完成 **新手引导关**，熟悉黄/蓝/红三种圈的操作与部署流程。

## 三、操作指南

### 3.1 通用操作

| 操作 | 说明 |
|------|------|
| **鼠标点击** | 菜单、按钮、棋盘放置、节奏单击 |
| **鼠标拖拽** | 部署阶段将手牌拖到棋盘 |
| **空格键** | 节奏阶段黄圈单击（与点击屏幕等效） |
| **长按** | 节奏阶段蓝圈：缩圈到达目标圆边缘后按住约 3 秒 |
| **连击** | 节奏阶段红圈：在 1.25 秒窗口内快速点击 |

**推荐环境：** Chrome 或 Edge 最新版，横屏窗口宽度 ≥ 960px 体验最佳。

### 3.2 节奏阶段

外圈从屏幕外缩入，当缩圈与中心 **目标圆** 边缘重合时进行操作。判定依据为 **几何重合**，而非单纯的时间偏差。

| 圈色 | 操作 | 判定要点 |
|------|------|----------|
| 🟡 **黄圈** | 重合时 **单击**（或按空格） | 完全重合 = Perfect；部分重叠 = Good；过早/过晚/未点 = Miss |
| 🔵 **蓝圈** | 重合后 **长按约 3 秒** | 按住至进度完成；松手过早可能 Miss |
| 🔴 **红圈** | 重合后 **连击** | 1.25 秒内 **6 次** Perfect 或 **4 次** Good |

**节奏结算：**

- 按本局准确度评定 **B → A → S → SS → SSS**
- 评级越高，本关发放道具 **数量越多**（后期关卡 SSS 最多约 21～25 个）
- 正式关节奏时长约 **60～105 秒**（随关卡递增）

### 3.3 部署阶段

| 操作 | 说明 |
|------|------|
| **放置** | 拖拽手牌到空格，或先点击手牌再点击目标格 |
| **合成** | 同类型、同阶且 **相邻** 的两个单位可合成升阶（最高 4 阶） |
| **合成次数** | 每局有限，约与道具数量相关，请优先合成关键路线 |
| **核心格** | 棋盘中央核心区域 **不可放置** 单位 |
| **查看属性** | 悬停或长按单位可查看 Tooltip（伤害、射程、职业等） |

**四轨道具与克制：**

| 道具 | 攻击方式 | 特点 |
|------|----------|------|
| 🥁 **节拍拳** | 近战单体 | 无法攻击飞行单位 |
| 🪘 **毒囊炮** | 范围溅射 | 可对空；克制护甲怪 |
| 🎵 **弧光术** | 弹射连锁 | 可对空；适合清群 |
| ✨ **凝霜律** | 近战减速 | 无法攻击飞行；可减速地面怪 |

> 飞行怪多的关卡（如 L05、L07）会 **偏向掉落毒囊炮 / 弧光术**，请预留对空位。

### 3.4 守卫阶段

| 项目 | 说明 |
|------|------|
| **胜利条件** | 所有波次出完且场上无敌人；L08 终局需 **击杀 Boss** |
| **失败条件** | 节拍核心 HP 降至 0 |
| **自动战斗** | 部署单位自动寻敌攻击，无需手动操作 |
| **音律币 ♪** | 击杀敌人、清空波次获得，用于修复核心 |
| **核心修复** | 消耗 ♪ 恢复核心 HP；每关修复次数有上限（L01～03：1 次；L04～06：2 次；L07：4 次；L08：5 次） |
| **通关星级** | 核心剩余 HP > 80% → 3★；> 50% → 2★；其余 → 1★ |

**敌人提示：**

- **飞行单位**（狮鹫等）：需毒囊炮 / 弧光术
- **护甲单位**（盾墙等）：毒囊炮溅射更有效
- **祝福牧师**：会治疗友军，建议优先击杀

### 3.5 主菜单与其他

| 功能 | 说明 |
|------|------|
| **选择关卡** | 按章节浏览，显示历史评级与星级 |
| **玩法说明** | 主菜单内可快速查看精简版规则 |
| **新手引导** | 首次进入可选教程关，分节奏 / 部署 / 守卫三阶段教练 |
| **存档** | 进度保存在浏览器 IndexedDB（`rhythm-guard-save`），换设备不同步 |

> 线上正式版隐藏「战斗模拟器」「Meta 升级」入口；本地 `localhost` 开发模式保留。

## 七、常见问题

**Q：节奏关没有声音？**  
A：部分浏览器需先点击画面以激活音频；教程关需点「知道了」后才开始 BGM。

**Q：部署时单位复制了？**  
A：已修复；请使用最新版。放置时勿重复点击同一空格。

**Q：飞行怪打不到？**  
A：节拍拳、凝霜律无法对空，请部署毒囊炮或弧光术。

**Q：存档丢失？**  
A：存档在浏览器本地，清除站点数据或换浏览器会重置。
