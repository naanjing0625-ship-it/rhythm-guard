# 游戏截图目录

将实机截图放在此目录，供 `docs/游戏说明.md` 引用。

## 推荐文件名

| 文件 | 场景 |
|------|------|
| `01-menu.png` | 主菜单 |
| `02-rhythm.png` | 节奏阶段 |
| `03-deploy.png` | 部署阶段 |
| `04-defense.png` | 守卫阶段 |
| `05-result.png` | 结算界面 |

## 采集方式

1. 浏览器打开游戏（线上或 `npm run dev`）
2. 进入对应界面后截图（建议 16:9，宽度 ≥ 1280px）
3. 保存为 PNG 到此目录

## 说明

AI 无法生成与实机一致的 Phaser 游戏画面，请使用真实游戏截图。
